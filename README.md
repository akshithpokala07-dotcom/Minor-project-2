Project Objective

This project was developed as part of the **Week 02 Minor Project** for the Unlox Web Development Program.

The objective of this project is to design and develop a responsive product landing page using HTML5 and CSS3 while applying modern layout techniques such as Flexbox and Responsive Design.

The website represents a fictional technology company called **Dina Tech**, showcasing two products:

- Dina Laptop
- Dina Phone

Project Objectives

- Create a responsive landing page
- Design a clean and user-friendly interface
- Practice HTML page structuring
- Use CSS3 for styling
- Implement Flexbox layouts
- Create responsive sections for different screen sizes

Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design
- Visual Studio Code

### How to Run

1. Download both the html and CSS file
2. Make sure to place them in a single folder
3. Open "index.html" in Visual Studio Code
4. Run using live server or open it directly into your browser.
